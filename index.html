FLEDGE DEMO

This demo shows a simple example of using FLEDGE to join an ad interest group and then initiate an on-device auction.

Browser Setting

You will need to use Chrome 91 or above run with the following flag:

--enable-features=FledgeInterestGroups,FledgeInterestGroupAPI

Starting DEMO

Go to the Advertiser sites below for joinInterestGroup() first

  • Shopping Site
  • Travel Site

then Go to the Publisher site to see Ads selected via runAdAuction().

  • Publisher Site
